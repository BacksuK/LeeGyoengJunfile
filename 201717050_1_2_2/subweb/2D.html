2D UNITY 분해(unity tutorials: 2D Roguelike)


(unity tutorials: 2D Roguelike)는 유니티 프로그램에서 기본적으로 참고자료가 될 수 있도록 무료로 제공해주는 게임입니다.
2D 게임의 구조와 스크립트를 살펴보도록 하겠습니다.

2D UNITY:sprite구조

유니티에서 UI,2D,3D 3분야에서 모두 사용되는 방법으로 최적화와 높은 편리성을 보여주는 기법입니다. 밑의 사진은 이 게임에서 sprite형태의 파일에서 원하는 이미지 뽑아서 이용하는 방식입니다.
위와 같은 방식으로 파일들을 이용하게 되면 한개의 이미지 파일에서 타일들을 가져오는 형식으로 게임 내부에서 인식받는 파일들이 적어지며 용량 또한 최적화 됩니다. 그리고 이미지 파일들을 sprite형식으로 묶을때 빈공간부터 채워넣기 때문에 쓸모없는 공간들을 없애므로 이미지 파일 자체의 용량또한 적어집니다.

2D UNITY:sprite Animation


스프라이트 이미지 구조는 애니메이션 기능을 구현할 수 있습니다.
이미지 파일에서 1프레임마다 재생될 파일들을 뽑아서 스크립트로 특정상황일때 어느 이미지가 출력되거나 만들어진 애니메이션 묶음을 재생시키는 식으로 사용됩니다.
스크립트문을 살펴보면 하늘색으로 처리되있는 부분이 바로 지정해둔 애니메이션 효과 파일을 가져오는 구문입니다. 위의 스크립트는 if문으로 특정 상황일시 어떠한 행동을 한다라는 것을 표현하는 것입니다.

2D UNITY:skeleton animation

유니티 2D애니메이션에서 3D애니메이션처럼 뼈대를 이용하는 유일한 방법입니다.

이 방법은 타 프로그램에서 2D이미지에 뼈대를 줘서 시간마다 부드럽게 움직이는 형태로 한파일로 여러가지의 애니메이션을 표현할 수 있으므로 고화질 게임에 많이 이용됩니다. 왼쪽의 자료들에서 예시를 보실 수 있습니다.
